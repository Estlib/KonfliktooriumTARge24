Meeskonna tutvustus


Tehke pisike sektsioon, kus tutvustate oma meeskonda. Lisate sinna lingid oma github õpimappidele,
ja ideele oma pontensiaalse lõputöö jaoks.

Iga meeskond teeb ka oma CSS stiili, stiilifaile on üks, kuid stiili peab rakendama nii,
et mitte kellegi teise stiili ei muudetaks.

Selle jaoks uuri siit: w3schools class selector
Kasuta järgnevat HMTL tabeli näidet, et teha oma meeskonna tabelike:

MEESKONNA NIMI
Meeskonnaliikmed: A B
